# Contributing compute

## Quick start

```bash
./scripts/run-profile.sh channel-delivery --issue 34093 --repo /path/to/openclaw
```

## Required evidence per issue

- OpenClaw commit SHA under test
- Profile name + command output
- PASS/FAIL verdict
- Log bundle checksum

## Verdict labels

- `reproducible`
- `not-reproducible`
- `needs-more-info`
- `duplicate/superseded`
